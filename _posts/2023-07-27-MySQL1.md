# MySQL 학습 1편 - 데이터베이스 개요 및 설치, 테이블 만들어보기



## 1. 시작하기 전

DB에 대해 배우려면 여러가지 관련 용어(DB, DBMS, SQL)들을 알아야 할 필요가 있다.

`데이터베이스`란 '데이터의 집합'이라고 할 수 있다. 이런 데이터베이스를 관리하고 운영하는 소프트웨어를 `DBMS(Database Management System)`라고 하는데, 이 DBMS에는 Oracle, PostgreSQL, 내가 배울 MySQL 등등 여러가지 DBMS가 존재한다. `SQL(Structed Query Language)`은 데이터베이스에서 사용되는 언어로, 데이터와 상호작용하기 위해서는 이 SQL을 익혀야 한다 .  



## 2. 설치 및 실행 

[https://dev.mysql.com/downloads/mysql/]: https://dev.mysql.com/downloads/mysql/	"mysql 다운로드 링크"

mysql 페이지로 가서 다운로드를 하고 터미널을 통해 실행한다. 

<img width="556" alt="스크린샷 2023-07-27 오후 10 09 05" src="https://github.com/eeaaomk98/eeaaomk98.github.io/assets/134247168/b9c23fd0-50d4-4ed0-b099-82d8ac7fddb3">

명령어를 입력 한 후 설치 할 때 설정한 비밀번호를 통해 접속하면 터미널의 하단 부분이 `mysql>`로 바뀌며 실행되는 모습을 볼 수 있다.



<img width="1136" alt="image" src="https://github.com/eeaaomk98/eeaaomk98.github.io/assets/134247168/057b2912-a7d8-4821-8d4b-d56d4e0cb6c6">

`MySQL Workbench`또한 설치해 주었다. `MySQL Workbench`는 MySQL을 이용하기 위한 GUI 클라이언트 도구이다. 



설치를 다 마쳤다면, 데이터베이스를 한 번 생성해보자. 데이터베이스를 생성하는 쿼리문은 다음과 같다.

```sql
CREATE DATABASE 데이터베이스이름;
```

`CREATE DATABASE`명령어를 사용하여 pet_shop 데이터베이스와 SlimeStore 데이터베이스를 생성하였다.



<br>

데이터베이스를 생성했다면, 생성된 데이터베이스를 확인해보자.

```sql
SHOW DATABASES;
```

<br>

MySQL Workbench에서도 chicken_coop 데이터베이스를 생성해주었고, Schemas탭에서 생성된 데이터베이스 목록을 확인할 수 있다. 

<img src="https://github.com/eeaaomk98/eeaaomk98.github.io/assets/134247168/73674503-748b-4402-a68f-ab945bc86a45" alt="image" style="zoom:150%;" />



<br>



<br>



만든 데이터베이스를 삭제 할 수도 있다.

```sql
DROP DATABASE 데이터베이스이름;
```



<br>



데이터 베이스를 생성했다면, `USE 데이터베이스이름;` 명령어를 통해 사용할 데이터베이스를 결정해주자.

```sql
USE 데이터베이스이름;
```





<br>

`SELECT database()`명령어는 현재 사용중인 데이터베이스를 보여주는 명령어이다.

```sql
SELECT database();
```

MySQL Workbench에서는 `USE chicken_coop;`명령어 대신 사용할 데이터베이스를 더블클릭 해주면 된다.



<br>

## 3. 테이블(TABLE)

관계형데이터베이스는 테이블들의 집합이다.

테이블은 데이터를 담는 그릇이다. 데이터베이스 안에서 구조화된 형식으로 관련된 데이터 컬렉션을 담고 있다. 

테이블은 열(heading)과 행(실제 데이터)으로 이루어져 있다. 각 열에는 데이터 타입을 설정해주어야 하는데,

보통 숫자로는 int형을, 문자열은 VARCHAR형을 써주어 최대 문자열의 길이를 정해줄 수 있다.



<br>



## 3-1. 테이블 생성하기

터미널에서 테이블 이름을 지정하고 모든 열의 이름과 그에 상응하는 데이터 타입을 지정해보자.

<img src="https://github.com/eeaaomk98/eeaaomk98.github.io/assets/134247168/1cc0b8d3-a045-4878-9a85-d0231410f05f" alt="스크린샷 2023-08-02 오전 11 48 02" style="zoom:200%;" />

`pet_shop` 데이터베이스에 `cats`라는 테이블을 생성해주었고, `name` 열에는 데이터 타입을 `VARCHAR`로 설정해서 최대 문자열을 50자리로 제한해두었고, `age`열에는 데이터 타입을 `INT`로 설정해주었다. 

<br>



<img src="https://github.com/eeaaomk98/eeaaomk98.github.io/assets/134247168/04dc3754-2032-4e6c-bd17-9d689938a012" alt="스크린샷 2023-08-02 오전 11 49 53" style="zoom:200%;" />

MySQL Workbench에서도 동일한 방식으로 테이블을 생성할 수 있다.



<br>

생성한 테이블을 보여주는 명령어는 다음과 같다.

```sql
SHOW TABLES;
```

<img src="https://github.com/eeaaomk98/eeaaomk98.github.io/assets/134247168/21556534-6ab2-4211-858c-d5e69c8fc3ac" alt="스크린샷 2023-08-02 오전 11 54 53" style="zoom:200%;" />



테이블의 세부사항을 보여주는 명령어는 다음과 같다.

```sql
SHOW COLUMNS FROM 테이블이름;
          or
DESC 테이블이름;
```



<img src="https://github.com/eeaaomk98/eeaaomk98.github.io/assets/134247168/1bbabc1a-68dd-4e25-bb17-0e2355bcc100" alt="스크린샷 2023-08-02 오전 11 56 39" style="zoom:150%;" />

`cats`테이블의 열의 이름, 데이터 타입 등을 보여준다.



<br>

## 4. 주석 달기

실행할 코드가 아닌 코드에 대한 정보나 메모를 남기고 싶은 경우 주석을 달아주면 된다.

```sql
-- 테이블을 보여주는 명령어
SHOW TABLES;
```

`-- `를 앞에 붙여주면 주석으로 사용할 수 있다.
